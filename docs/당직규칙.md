# 당직 규칙

## 현재 상태 (2026-07-20 ~)

**오후2 당직 폐지.** 2026년 7월 20일 배정분부터 오후2 슬롯을 사용하지 않는다.
다시 운영될 수도 있는 정책이라 언제 재개될지는 정해지지 않았다. 코드(`lib/print-config.js`)의
`dutySlots`/`printConfig`에는 오후2 정의를 그대로 남겨뒀다 — 재개 시 바로 쓸 수 있도록 유지하는
것으로, 실수로 지운 것이 아니다. 다만 폐지 기간 중에는 `duties` 테이블에 오후2 행을 만들지 않는다.

## 슬롯 정의 (오후2 운영 시 기준)

| 슬롯 | 근무시간 | 비고 |
|---|---|---|
| 오전1 | 7:30–16:30 | 당일 조기퇴근 1.5시간 |
| 오전2 | 8:00–17:00 | 당일 조기퇴근 1.0시간 |
| 오후1 | 9:00–18:30 | 다음 근무일 0.5시간 일찍 퇴근 |
| 오후2 | 9:30–19:30 | 1시간 시간외수당 지급 (폐지 중) |

당직 커버 시간대(실제 당직 업무가 필요한 시간): 오전1(7:30–9:00), 오전2(8:00–9:00),
오후1(16:30–18:30), 오후2(16:30–19:30, 폐지 중).

비고: "보육교사 - 1개월 단위 탄력적 근로제" 적용.

## 조기퇴근 파생 규칙

당직 배정에서 자동으로 계산됨 (`lib/calendar.js`의 `deriveEarlyLeave`):

- 오전1 당직 → **당일** 1.5시간 조기퇴근
- 오전2 당직 → **당일** 1.0시간 조기퇴근
- 오후1 당직 → **다음 근무일**(주말·공휴일 제외) 0.5시간 조기퇴근
- 오후2 당직 → 조기퇴근 없음, 대신 1시간 시간외수당 (현재 폐지 중이라 해당 없음)

조기퇴근 시간 → 실제 퇴근시각: 0.5시간→17:30, 1시간→17:00, 1.5시간→16:30, 2시간→16:00

## 휴일/주말 처리

- 일요일은 당직표에서 제외
- 토요일은 당직 배정이 있어도 표시하지 않음(대시 처리)
- 공휴일은 별도 표기

## 배정(로테이션) 방식

코드에 자동 배정/로테이션 알고리즘은 없다. `duties` 테이블에 수동으로 입력된 데이터를
그대로 화면·인쇄물에 반영할 뿐이다.
